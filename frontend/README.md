# SmartEMI Planner – Frontend (Week 4)

This week focuses on building a simple frontend dashboard to interact with the backend APIs.

### ✅ Features Implemented
- Basic UI for user interaction.
- Two buttons to:
  - Fetch Financial Summary.
  - Optimize EMI.
- Displays data returned by backend in a formatted JSON view.

### 🧠 Backend Integration
The frontend calls:
- `http://127.0.0.1:5000/get_summary`
- `http://127.0.0.1:5000/optimize`

### ⚙️ Usage
1. Run backend server:
2. Open `frontend/index.html` in browser.
3. Click buttons to see real-time backend data.
